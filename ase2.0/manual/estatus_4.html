Acciones para Estatus Asignado

La OT ya tiene Operadores Asignados para cada SOT, ahora el Supervisor de cada Área debe imprimir las SOTs para entregarlas a los Operadores correspondientes; también es opción del Jefe de Taller o Controlista imprimir las SOTs para entregarlas al Supervisor de cada Área para su repartición a Operadores.
Muestra los detalles de las Sub Ordenes de Trabajo o Tareas que pertenecen a esta Orden de Trabajo.
Muestra los documentos e imágenes que pertenecen a esta Orden de Trabajo.
Permite agregar nuevos documentos o imágenes a la Orden de Trabajo.
